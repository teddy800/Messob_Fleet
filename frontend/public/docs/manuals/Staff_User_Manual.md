## 1. Introduction

### 1.1 Purpose of This Manual

This manual guides staff members through using the MESSOB Fleet Management System to request and track institute vehicles for official purposes.

### 1.2 What You Can Do

As a **Staff User**, you can:
- ✅ Create trip requests for official travel
- ✅ View status of all your requests
- ✅ Track your assigned vehicle in real-time
- ✅ Cancel pending requests if plans change
- ✅ View your request history

### 1.3 System Requirements

- **Browser:** Chrome, Firefox, Safari, or Edge (latest version)
- **Internet:** Stable connection for real-time tracking
- **Device:** Desktop, laptop, tablet, or smartphone

## 3. Creating Trip Requests

### 3.1 Starting a New Request

1. Click **"New Request"** in the sidebar
2. The 4-step wizard will guide you through

### 3.2 Step 1: Trip Details

**Purpose/Justification** (Required)
- Enter detailed reason for the trip
- Minimum 10 characters required
- Be specific: "Attend procurement meeting at Ministry of Finance"

**Vehicle Category** (Required)
Select based on your needs:
- **Sedan:** 1-4 passengers, city travel
- **SUV:** Rough terrain, remote locations
- **Pickup:** Cargo transport
- **Bus:** 8+ passengers
- **Mini-Bus:** 5-8 passengers

**Example:**
```
Purpose: Attend quarterly budget review meeting with Ministry of Finance officials
Vehicle: Sedan (suitable for 2 staff members, city travel)
```

Click **"Next"** to continue.

### 3.3 Step 2: Schedule

**Start Date & Time**
- Select when you need the vehicle
- Cannot select past dates/times
- Consider traffic and travel time

**End Date & Time**
- When you'll return the vehicle
- Must be after start time
- Add buffer time for delays

**Multi-Day Trips:**
- System supports trips spanning multiple days
- Select appropriate end date

**Duration Display:**
- System automatically calculates trip duration
- Review to ensure it's correct

Click **"Next"** to continue.

### 3.4 Step 3: Locations

**Pickup Location**
- Where the vehicle should pick you up
- Use autocomplete for known locations
- Or click on map to set precise location
- Default: Your office location

**Destination**
- Where you're traveling to
- Use autocomplete or map selection
- Be specific (building name, address)

**Tips:**
- ✅ Use full addresses for accuracy
- ✅ Include landmarks if needed
- ✅ Verify location pins are correct on map

Click **"Next"** to continue.

### 3.5 Step 4: Review & Submit

**Review All Details:**
- ✓ Purpose is clear and complete
- ✓ Dates and times are correct
- ✓ Locations are accurate
- ✓ Vehicle category matches your needs

**Actions:**
- **Back:** Return to previous step to edit
- **Save as Draft:** Save without submitting
- **Submit for Approval:** Send to dispatcher

**After Submission:**
- Request status changes to "Pending"
- Dispatcher will be notified
- You'll receive email confirmation
- Typical review time: 24 hours

## 4. Managing Your Requests

### 4.1 Viewing All Requests

Click **"My Requests"** to see:
- All your trip requests
- Current status of each
- Quick action buttons

### 4.2 Understanding Request Status

**Status Colors:**
- 🟡 **Draft (Yellow):** Saved but not submitted
- 🟠 **Pending (Orange):** Awaiting dispatcher review
- 🟢 **Approved (Green):** Vehicle and driver assigned
- 🔵 **In Progress (Blue):** Trip is active
- ⚫ **Completed (Gray):** Trip finished
- 🔴 **Rejected (Red):** Request declined
- ⚪ **Closed (White):** Archived

### 4.3 Viewing Request Details

**Click on any request to see:**
- Full trip information
- Assigned vehicle (if approved)
- Assigned driver (if approved)
- GPS tracking (if in progress)
- Status history with timestamps

### 4.4 Canceling a Request

**For Pending Requests:**
1. Open the request
2. Click **"Cancel Request"**
3. Confirm cancellation
4. Status changes to "Draft"

**⚠️ Important:**
- Can only cancel "Pending" requests
- Cannot cancel approved or in-progress trips
- Contact dispatcher for changes to approved trips

### 4.5 Editing a Request

**Saved Drafts:**
- Open the draft
- Click "Edit"
- Make changes
- Re-submit

**Pending/Approved:**
- Cannot edit directly
- Must cancel and create new request
- Or contact dispatcher for modifications

## 5. Live GPS Tracking

### 5.1 Accessing Live Tracking

**For Approved/In-Progress Trips:**
1. Go to **"My Requests"**
2. Click on trip with "Approved" or "In Progress" status
3. Click **"Track Vehicle"** button
4. Map view opens with live position

### 5.2 Understanding the Map

**Map Elements:**
- 📍 **Blue Pin:** Your vehicle's current location
- 🟢 **Green Pin:** Pickup location
- 🔴 **Red Pin:** Destination
- **Blue Line:** Planned route
- **Gray Line:** Path already traveled

**Vehicle Information Panel:**
- Plate number
- Current speed
- Direction (heading)
- Last update time
- Driver name and contact

### 5.3 Map Controls

**Zoom:**
- Click **+** to zoom in
- Click **-** to zoom out
- Use mouse wheel

**Pan:**
- Click and drag map
- Use arrow keys

**Center on Vehicle:**
- Click **"View Progress"** button
- Map centers on current position

**Refresh:**
- Position updates automatically every 1 second
- No manual refresh needed

### 5.4 Collaborative Features

**Shared Rides:**
If multiple staff share the same vehicle:
- You'll see other passengers' pickup points
- Contact information (if allowed)
- Helps coordinate meeting at pickup

**Dynamic Pickup:**
- Adjust your exact pickup point if needed
- Click and drag your pin
- Driver sees updated location immediately

## 7. Tips & Best Practices

### 7.1 Request Creation Tips

**✅ DO:**
- Submit requests at least 48 hours in advance
- Provide detailed, clear justifications
- Double-check dates and times
- Add buffer time for delays
- Select appropriate vehicle category

**❌ DON'T:**
- Submit last-minute requests
- Use vague purposes ("meeting")
- Schedule back-to-back trips
- Request larger vehicles than needed

### 7.2 Maximizing Approval Chances

1. **Plan Ahead:** Early requests have higher priority
2. **Be Specific:** Clear purpose helps dispatchers
3. **Flexibility:** Indicate if timing is flexible
4. **Urgency:** Mark truly urgent trips appropriately

### 7.3 During Your Trip

**Before Departure:**
- ✅ Verify pickup time and location with driver
- ✅ Share your phone number with driver
- ✅ Be ready 10 minutes early

**During Travel:**
- ✅ Keep phone accessible
- ✅ Notify driver of any route changes
- ✅ Report any issues immediately

**After Completion:**
- ✅ Confirm trip completion with driver
- ✅ Provide feedback if requested

## 8. Troubleshooting

### 8.1 Cannot Log In

**Problem:** Invalid credentials error

**Solutions:**
1. Verify email is correct (no extra spaces)
2. Check Caps Lock is OFF
3. Try "Forgot Password" link
4. Clear browser cache and cookies
5. Contact IT support if still failing

### 8.2 Request Not Submitting

**Problem:** Submit button doesn't work

**Solutions:**
1. Ensure all required fields are filled
2. Check dates are valid (not in past)
3. Verify purpose has 10+ characters
4. Try different browser
5. Check internet connection

### 8.3 GPS Tracking Not Working

**Problem:** Vehicle not showing on map

**Solutions:**
1. Verify trip status is "Approved" or "In Progress"
2. Refresh the page
3. Check internet connection
4. Ensure GPS device is active in vehicle
5. Contact dispatcher if issue persists

### 8.4 Cannot Cancel Request

**Problem:** Cancel button is disabled

**Reason:**
- Can only cancel "Pending" requests
- Approved trips must be canceled by dispatcher

**Action:**
- Contact dispatcher via phone or email
- Explain reason for cancellation

## 9. FAQ

**Q1: How long does approval take?**  
A: Most requests are reviewed within 24 hours. Urgent requests are prioritized.

**Q2: Can I request a specific vehicle?**  
A: No, dispatchers assign vehicles based on availability and suitability.

**Q3: Can I request a specific driver?**  
A: Driver assignment is done by the dispatcher based on schedules.

**Q4: What if my trip runs longer than scheduled?**  
A: Contact the dispatcher immediately to extend the booking.

**Q5: Can I see who else is using the same vehicle?**  
A: Yes, if you share a vehicle, you'll see other passengers' pickup points.

**Q6: What if the driver is late?**  
A: Contact the driver directly (number shown in trip details) or call dispatcher.

**Q7: Can I modify an approved request?**  
A: Contact dispatcher - they can modify approved bookings.

**Q8: How do I report an issue during my trip?**  
A: Contact dispatcher immediately: +251 11 XXX XXXX

**Q9: Can I use the system on my phone?**  
A: Yes, the system is mobile-responsive and works on all devices.

**Q10: What happens to my pending request if rejected?**  
A: You'll receive notification with reason. You can create a new request.
